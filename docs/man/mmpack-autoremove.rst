=================
mmpack-autoremove
=================

-------------------
autoremove packages
-------------------

SYNOPSIS
========

``mmpack autoremove`` [-h|--help] autoremove

DESCRIPTION
===========
**mmpack autoremove** remove all the packages that have not been manually
installed such that no manually installed packages depend on them directly or
indirectly.

OPTIONS
=======
``-h|--help``
  Show help and exit

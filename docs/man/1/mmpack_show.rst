===============
mmpack_show
===============

-------------
show packages
-------------

:Author: Gabriel Ganne <gabriel.ganne@mindmaze.ch>,
         Nicolas Bourdaud <nicolas.bourdaud@mindmaze.ch>
:Date: 2018-10-25
:Manual section: 1

SYNOPSIS
========

**mmpack show** [-h|--help] *package*

DESCRIPTION
===========
**mmpack show** show given package metadata.

OPTIONS
=======
--help, -h
  Show help and exit

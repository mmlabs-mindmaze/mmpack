mmpack_search
===============

----------------
search packages
----------------

:Author: Gabriel Ganne <gabriel.ganne@mindmaze.ch>,
         Nicolas Bourdaud <nicolas.bourdaud@mindmaze.ch>
:Date: 2018-10-25
:Manual section: 1

SYNOPSIS
========

**mmpack search** [-h|--help] *pattern*

DESCRIPTION
===========
**mmpack search** searches given pattern within all the available package names.

OPTIONS
=======
--help, -h
  Show help and exit
